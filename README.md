Project From Scratch

This is a React app built using TypeScript and Vite, with a strong focus on code quality and clean structure.

Tech & Versions-

React (19.2.3)
TypeScript (5.5.4)
Vite (7.3.1) — used as bundler and dev server
Node.js (>=18)

Code Quality-

ESLint is used with AirBnB rules, TypeScript support, and React Hooks plugin. The project follows strict linting with zero warnings allowed.

Prettier is not used — ESLint handles all formatting and style rules.

How to Run-

npm install
npm run dev → runs app on http://localhost:3000

Other Commands-

npm run lint → check lint errors
